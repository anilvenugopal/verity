Navigating the Registry

How to find agents, tasks, prompts, tools, and models in the portal.

Registry section

Click Registry in the left navigation bar. The registry opens with a secondary nav showing Agents, Tasks, Prompts, Tools, and Models. Select the entity type you need.

Finding a specific item

Press Ctrl+J (or ⌘J on Mac) to open the command palette. Type the name of an agent, task, prompt, or tool — results appear grouped by type. Press Enter or click to navigate directly to the detail page.

Agent and task detail pages

The entity detail page shows all versions in a table. Click a version row to open the version detail, which shows the full composition manifest (prompts, tools, bindings, inference config) and the lifecycle stage.

Prompt and tool where-used

On a prompt or tool detail page, click Where used next to any version to see which agent or task versions reference it. This is useful for understanding blast radius before updating a shared component.

Champion badge

The champion badge on a version row indicates the currently live version for that executable. Only one version per executable can be champion at a time; promoting a new champion atomically revokes the previous one.
